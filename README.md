## 📂 Repository Structure

The repository is organized into three main modules to ensure a clear learning path and easy retrieval of solutions.

- **`Codeforces/`**: Core practice module.
  - `Solutions/`: Well-documented source code for solved problems (mainly in C/C++).
  - `Problem-Statements/`: Detailed problem descriptions and analysis.
  - `Contest-Records/`: Performance tracking and reflections on specific contest rounds.
- **`Knowledge-Base/`**: Theoretical foundation and learning notes.
  - `Algorithms/`: Insights into algorithmic strategies (DP, Greedy, DFS/BFS, etc.).
  - `Data-Structures/`: Implementations and notes on Trees, Graphs, Hash Tables, etc.
- **`Templates/`**: Reusable code snippets and standard implementations for quick deployment during contests.
